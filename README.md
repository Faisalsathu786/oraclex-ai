OracleX AI

Decentralized AI Prediction Market Platform on 0G Ecosystem

## Overview

OracleX AI is a production-grade, decentralized AI-powered prediction market platform where users can create markets, place predictions, analyze AI insights, compete on leaderboards, and earn rewards - all powered by the 0G ecosystem.

## Tech Stack

- Frontend: Next.js 15, TypeScript, Tailwind CSS, shadcn/ui, Framer Motion
- Smart Contracts: Solidity 0.8.24, Hardhat, OpenZeppelin Upgradeable
- Backend: Next.js API Routes, Prisma ORM, PostgreSQL
- AI: OpenAI / Claude API
- Infrastructure: 0G Chain, 0G Storage, 0G Compute
